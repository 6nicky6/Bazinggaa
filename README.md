# Bazingga — Claude Code Build Kit

## What this is
Everything Claude Code, Cowork, and Claude Chat need to build Bazingga end-to-end.

## Files
- `CLAUDE.md` — master context (Claude Code auto-reads this)
- `docs/SETUP_GUIDE.md` — one-time Mac setup (~30-40 min, copy-paste commands)
- `docs/FIRST_PROMPT.md` — Sprint 1 prompt + all future sprint prompts
- `docs/SUPABASE_SCHEMA.sql` — database, ready to paste
- `docs/ROADMAP.md` — build → market → grow → monetize
- `docs/PROGRESS_LOG.md` — update 3 lines per sprint

## Start here
1. Unzip this into `~/Bazingga` on your Mac
2. Follow `docs/SETUP_GUIDE.md` top to bottom
3. Run `claude` inside the folder, paste the Sprint 1 prompt
4. Report back to Claude Chat after each sprint

## Golden rules
- One sprint = one goal = 45 minutes
- Usage credits OFF in Claude settings (no company billing, ever)
- Scope is guarded by CLAUDE.md — if it's not MVP, it waits
